# Kesar website media

Add website media inside these folders:

- `videos/` — hero videos and other motion assets
- `images/` — hero posters, food photos and gallery images

The homepage hero currently uses the user-uploaded files:

- `videos/heropage-mandi.mp4`
- `images/heropage-mandi-4k.webp`

For a future replacement, upload the new files into the same folders and keep
these exact filenames. No code change is required.
