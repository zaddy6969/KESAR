# Kesar website media

Add website media inside these folders:

- `videos/` — hero videos and other motion assets
- `images/` — hero posters, food photos and gallery images

The homepage hero currently uses:

- `videos/kesar-mandi-hero.mp4`
- `images/kesar-mandi-hero-poster.webp`

To replace the hero later, upload a new MP4 using the exact filename
`kesar-mandi-hero.mp4`. No code change is required.
